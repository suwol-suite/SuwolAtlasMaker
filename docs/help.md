# In-App Help

Suwol Atlas Maker includes an in-app guide under Help > Guide.

Guide tabs:

- Quick Start
- Basic Usage
- Using with Unity
- Using with MonoGame
- Troubleshooting
- File Guide

All guide text lives in the `help` i18n namespace for English and Korean.
The guide is renderer UI only. It does not affect project files or atlas JSON.

Help > Troubleshooting opens the same guide directly on the troubleshooting
tab.

Help > Check for Updates opens the Status panel and checks for Linux AppImage
updates when the app is running as a packaged Linux AppImage. Other platforms
and development builds report that automatic updates are unsupported.
